
# 📊 Bluestock Mutual Fund Analytics Capstone

## 📌 Project Overview

This project is an end-to-end Mutual Fund Analytics Capstone built using Python, SQL, SQLite, Power BI, and Financial Analytics techniques.

The objective is to analyze Indian Mutual Fund data, evaluate fund performance, understand investor behavior, build risk analytics, and create an interactive dashboard for business insights.

---

## 🎯 Objectives

- Perform ETL and data cleaning
- Build a SQLite database
- Conduct Exploratory Data Analysis (EDA)
- Compute financial performance metrics
- Perform advanced risk analytics
- Build an interactive Power BI dashboard
- Generate business insights and recommendations

---
